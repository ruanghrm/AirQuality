# Mapeamento Climático de Belém

## Descrição
Este projeto visa criar um mapa interativo que mostra a qualidade do ar em diferentes bairros de Belém.

## Tecnologias Usadas
- HTML
- CSS
- JavaScript
- Google Maps
- GeoJSON
